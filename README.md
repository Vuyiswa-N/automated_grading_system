The Automated Answer Marking System is a web-based application built using Flask that is designed to automatically grade short answers for Grade 12 CAPS Physical Sciences. The system takes two text files as input, a student's answer and a memorandum and uses Natural Language Processing (NLP) techniques to compare them and produce a similarity-based grade. The preprocessing pipeline cleans the text by converting it to lowercase, removing punctuation, filtering out common stopwords, and lemmatizing domain-specific Physics and Chemistry terminology to ensure that words like "forces" and "force" or "accelerating" and "accelerate" are treated as the same term. Once preprocessed, the system generates n-gram models such as bigrams (two-word phrases), and trigrams (three-word phrases) and applies a custom TF-IDF vectorizer to each, which weighs terms based on how important they are relative to both documents. Cosine similarity is then calculated for each n-gram model separately, measuring the angle between the two TF-IDF vectors to produce a percentage score indicating how similar the student's answer is to the memorandum. The bigram and trigram scores are then fused together through an equal weighted average to produce a final estimated grade. The results are displayed on a dedicated results page showing a comparison table of all three n-gram scores alongside the fusion grade, which either marks the answer as a proposed grade if it scores 50% or above, or flags it for human evaluation if it falls below 50%.
